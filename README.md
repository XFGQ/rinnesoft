# RinneSoft Portfolio

Furkan Duman ve Alper Altınpınar'a ait kişisel portföy sitesi. Projelerimizi, yeteneklerimizi ve iletişim bilgilerimizi içerir.

## Teknolojiler

- Java 21 + Spring Boot 3.2
- MySQL / MariaDB
- Maven

## Kurulum

```bash
# Bağımlılıkları yükle ve çalıştır
./mvnw spring-boot:run
```
